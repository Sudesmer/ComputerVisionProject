# ComputerVisionProjesi

Bilgisayar gorusu tabanli cizim ve kayit deneyleri iceren Python projesi.

## Icerik

- `ressam.py`: Ana uygulama akisi
- `kamera_test.py`: Kamera test scripti
- `assets/`: Muzik ve diger medya varliklari
- `captures/`: Kaydedilen cizim goruntuleri
- `exports/`: Oturum export ciktilari

## Kurulum

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

> Not: `requirements.txt` yoksa mevcut paketleri `pip freeze > requirements.txt` ile olusturabilirsin.

## Calistirma

```bash
python3 ressam.py
```
